# Описание проекта

Данный анализ выполнен для некоммерческого проекта **«Карта ДТП»** ([dtp-stat.ru](http://dtp-stat.ru/)), который занимается открытой аналитикой данных по безопасности дорожного движения в России.

Цель проекта:
 - изучение динамики дорожно-транспортных происшествий за доступный период,
 - выявление ключевых закономерностей и факторов риска, влияющих на тяжесть и частоту аварий, 
 - проверка гипотез для статистического подтверждения наблюдаемых тенденций.

---

# Навыки

- **Веб-скрапинг** и автоматизированная загрузка файлов  
- Загрузка и нормализация **GeoJSON данных**  
- **Создание отдельных таблиц** для ДТП, участников и транспортных средств  
- Экспорт очищенных DataFrame в **базу данных SQLite**  
- Исследовательский анализ данных с использованием **Data Profiling**  
- Очистка и преобразование данных, конвертация типов и создание новых признаков  
- Обработка аномалий  
- Анализ взаимосвязей с помощью **SQL-запросов**  
- Визуализация данных  
- **Проверка гипотез с помощью теста хи-квадрат**

---

# Инструменты

Python, Pandas, NumPy, re, Scipy, Matplotlib, Seaborn, Requests, Zipfile, JSON, OS, SQLite3, BeautifulSoup, ydata_profiling, Google Colab / Jupyter Notebook.

# Результаты

- Очищенные и нормаоизованные данные
- Созданы 3 отдельные таблицы: инценденты, участники и транспортные средства, сохранённые в базе SQLite
- Проведен глубокий анализ предоставленных данных
- Выявлены ключевые закономерности и факторы риска ДТП  
- Статистичеки подтверждены три гипотезы
- Формирование надёжной базы данных для дальнейших исследований и принятия решений в области безопасности дорожного движения
